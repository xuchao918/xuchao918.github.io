问题1：浏览器URL地址自动加上#号

浏览器访问VUE项目会在访问地址后面加上#，这个#其实是VUE的HASH模式所产生的，正确点来说是因为VUE使用了HASH模式。

如果不想有#应该怎么做：修改路由Router的mode为history即可
例如在vue init webpack my-project创建项目完毕以后，在src->router->index.js里修改
